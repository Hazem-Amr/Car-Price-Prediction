# Car Price Prediction

## Project Overview

A full-stack AI-powered car price prediction system. Enter vehicle details (brand, model, mileage, power, etc.) and get an instant price estimate powered by a CatBoost model (R² ≈ 0.94).

## Architecture

| Layer | Stack | Description |
|-------|-------|-------------|
| **Frontend** | React + Vite + Tailwind v4 | Dark glassmorphic UI with prediction form |
| **Backend** | FastAPI + Uvicorn | REST API serving predictions via `/predict` |
| **ML Model** | CatBoost Regressor | Trained on `Cars_Data.csv` with feature engineering |

## How to Run

### Prerequisites

- **Python 3.10+** with `pip`
- **Node.js 18+** with `npm`

### 1. Train the Model (one-time)

```bash
cd backend
pip install -r requirements.txt
python train.py
```

This generates `model.pkl` and `features.pkl`. You should see output like:

```
=== Evaluation ===
MAE:  167,520.59
R²:   0.9388
```

### 2. Start the Backend

```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```

The API will be available at `http://localhost:8000`. Test it:

```bash
# Health check
curl http://localhost:8000/health

# Predict
curl -X POST http://localhost:8000/predict \
  -H "Content-Type: application/json" \
  -d '{"brand":"ford","model":"Kuga","color":"black","registration_year":2018,"power_ps":140,"fuel_type":"Petrol","transmission_type":"Automatic","fuel_consumption":6.5,"mileage":50000}'
```

### 3. Start the Frontend

```bash
cd frontend
npm install
npm run dev
```

Open **http://localhost:5173** in your browser, fill in the car details, and click **⚡ Predict Price**.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/health` | Server status & model load check |
| `GET` | `/options` | Dropdown values (brands, colors, fuel types, transmissions) |
| `POST` | `/predict` | Predict car price from input features |

## Dependencies

**Backend:** `fastapi`, `uvicorn`, `catboost`, `joblib`, `numpy`, `pandas`, `scikit-learn`
**Frontend:** `react`, `tailwindcss`, `vite`
